# RSA Cipher

Простой проект на Python, реализующий алгоритм RSA. Используются простые числа не более 10^6 для генерации ключей.

## Что умеет
- Генерация открытого и закрытого ключей
- Шифрование и расшифровка текста
- Интерактивный режим
- Юнит-тесты

## Требования
- Python 3.7 или выше
